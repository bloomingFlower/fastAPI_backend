# FastAPI Backend

- 얼마나 쉽게 endpoint를 만들 수 있나 확인하기 위해 이것 저것 만들기
### API 구현
### 1. 도커 이미지 빌드 
`docker build -t fastAPIend .`
### 2. 도커 실행 
`docker-compose up -d --build`
### 3. API 테스트 페이지
`http://localhost:8000/static/test.html`

---
## System Requirements
- Docker
- Docker-compose

## Framework & DB
- FastAPI (Python)
- PostgreSQL DB
